# Portail

Une PWA immersive : ouvre une fenêtre vers un autre univers en 3D, exploré au mouvement de ton téléphone (gyroscope).

- 100% local et hors-ligne, aucune donnée envoyée
- Three.js embarqué en local, aucun code distant
- Installable sur Android (PWA)

## Utilisation
Sur téléphone : bouge l'appareil dans tous les sens pour regarder autour de toi.
Sur ordinateur : glisse le doigt ou la souris.

En ligne via GitHub Pages.
